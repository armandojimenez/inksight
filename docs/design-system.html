Design System

Tokens, components, and accessibility patterns for the Inksight visual assistant.

Color Palette

Derived from Inkit's brand blue. Every color includes its WCAG contrast ratio against white.

Typography

Three font families from Inkit's stack — Space Grotesk for display, Archivo for body, Space Mono for code.

Spacing Scale

4px base unit on an 8px grid. Used for margins, padding, and gaps throughout the system.

Border Radius

Progressive rounding from subtle to circular.

Shadows

Blue-tinted shadows matching the neutral palette. Five levels of elevation.
